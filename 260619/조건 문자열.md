# 📌 문제 이름

프로그래머스 181934번 - 조건 문자열

## 🔗 문제 링크

## 💡 문제 해결 전략

딱 봐도 케이스를 4개로 나눠야 할 것으로 보임
조금이라도 덜 귀찮기 위해 ineq과 eq를 합치자
I love 중간 리턴
I love 삼항연산자

## 💻 코드

```java
class Solution {
    public int solution(String ineq, String eq, int n, int m) {
        String factor = ineq + eq;

        if (factor.equals(">=")) {
            return n >= m ? 1 : 0;
        }

        if (factor.equals("<=")) {
            return n <= m ? 1 : 0;
        }

        if (factor.equals(">!")) {
            return n > m ? 1 : 0;
        }

        if (factor.equals("<!")) {
            return n < m ? 1 : 0;
        }

        return 0;
    }
}
```
