# 📌 문제 이름

프로그래머스 181935번 - 홀짝에 따라 다른 값 반환하기

## 🔗 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/181935

## 💡 문제 해결 전략

일단 케이스가 2개다
그리고 그 안에서 또 짝수인지 아닌지 조건문이 필요하다
영한 씨가 쓰지 말라고 했던 if 문 뒤에 {} 안 쓰기를 해보자

## 💻 코드

```java
class Solution {
    public int solution(int n) {
        if (n % 2 == 0) {
            int sum = 0;

            for (int i = 1; i <= n; i++) {
                if (i % 2 == 0) sum += i*i;
            }

            return sum;
        }

        int sum = 0;

        for (int i = 1; i <= n; i++) {
            if (i % 2 == 1) sum += i;
        }

        return sum;
    }
}
```
