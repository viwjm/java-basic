# 📌 문제 이름

큐 만들기

## 🔗 문제 링크

## 💡 문제 해결 전략

일단은 메서드들을 형태만 다 만들어 놓자
push, empty는 boolean, size는 int를 반환해야 함이 자명하다
pop, front, back은 애매하다 int를 반환해야 하기는 하는데 그러면 큐가 empty 한 경우를 표현할 수 없다
pop, front, back은 항상 empty 검사를 한 후 사용하면 되긴 하지만 적절한 제약으로 막아두는 것이 더 안전할 것이다
그런데 문제를 다시 보니 어차피 정수는 1 이상이라고 한다 그냥 -1 을 실패로 사용하면 되겠다
empty 도 boolean 이 아니라 그냥 1, 0을 출력하도록 하면 되겠지만 왠지 그냥 boolean을 사용하고 싶다

첫 요소 인덱스와 마지막 요소 인덱스가 있어야 한다
그런데 생각해보면 마지막 요소 인덱스 + 1 값을 가지고 있어야 한다
그래야 그냥 empty 한 경우와 하나가 들어 있는 경우를 구분할 수 있다
그런데 문제가 있다 이러면 또 큐가 비어 있는 것과 꽉 찬 것이 구분이 안 된다(인덱스가 4가 넘으면 다시 0으로 가도록 구현할 것이기 때문)
그냥 isEmpty 상태값을 만들어서 따로 추적하자
이러면 마지막 요소 인덱스 + 1이 아니라 그냥 마지막 요소 인덱스를 추적해도 되므로 훨씬 생각하기 편해진다
그리고 empty 메서드가 필요가 없어진다
그런데 메서드로 만드려다 보니 int형인 인덱스는 메서드 안에서 값 변경이 안 된다
애초에 메서드로 만드려고 한 것이 헛된 꿈이었던 것 같다 메서드는 클래스를 배우고 나서 쓰도록 하자

명령은 종류가 두 개이다
push 만큼은 push-띄어쓰기-정수 의 형태로 나와야 한다
나머지는 그냥 명령어만 들어가면 된다 equals 로 하면 된다는 뜻이다
isPush 만큼은 메서드를 따로 만들자
number로 변환 가능한지 확인하는 방법이 기억이 안 나서 일단 try catch로 했는데 별로인 것 같다

인덱스가 4가 넘으면 다시 0으로 가도록 구현하기 위해서 % 5 를 사용하는 것 중요하다
그리고 isEmpty로 정말 비었나 검사하고(인덱스만으로는 알 수가 없다) isEmpty 값을 적절하게 수정해주는 것 역시 중요하다

## 💻 코드

```java
public class QueueMain {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int arraySize = 5;
        int[] queue = new int[arraySize];
        boolean isEmpty = true;
        int firstIndex = 0;
        int lastIndex = 0;

        System.out.println("명령 수를 입력하세요:");
        int numOrders = scanner.nextInt();
        scanner.nextLine();

        for (int i = 0; i < numOrders; i++) {
            String order = scanner.nextLine();

            if (isPush(order)) {
                if (isFull(firstIndex, lastIndex, arraySize)) {
                    System.out.println("배열 꽉 참");

                    continue;
                }

                int number = Integer.parseInt(order.split(" ")[1]);

                if (isEmpty) {
                    queue[lastIndex] = number;
                    isEmpty = false;
                } else {
                    lastIndex = (lastIndex + 1) % arraySize;
                    queue[lastIndex] = number;
                }

                continue;
            }

            if (order.equals("pop")) {
                if (isEmpty) {
                    System.out.println("-1");

                    continue;
                }

                System.out.println(queue[firstIndex]);
                queue[firstIndex] = 0;

                if (firstIndex == lastIndex) {
                    isEmpty = true;
                } else {
                    firstIndex = (firstIndex + 1) % arraySize;
                }

                continue;
            }

            if (order.equals("size")) {
                if (isEmpty) {
                    System.out.println("0");

                    continue;
                }

                int size = firstIndex <= lastIndex ? lastIndex - firstIndex + 1 : lastIndex + 5 - firstIndex + 1;
                System.out.println(size);

                continue;
            }

            if (order.equals("empty")) {
                if (isEmpty) {
                    System.out.println("1");

                    continue;
                }

                System.out.println("0");

                continue;
            }

            if (order.equals("front")) {
                if (isEmpty) {
                    System.out.println("-1");

                    continue;
                }

                System.out.println(queue[firstIndex]);

                continue;
            }

            if (order.equals("back")) {
                if (isEmpty) {
                    System.out.println("-1");

                    continue;
                }

                System.out.println(queue[lastIndex]);

                continue;
            }

            System.out.println("잘못된 입력입니다");
        }
    }

    private static boolean isPush(String order) {
        String[] orderComponents = order.split(" ");

        if (orderComponents.length != 2) {
            return false;
        }

        try {
            Integer.parseInt(orderComponents[1]);
        } catch (NumberFormatException e) {
            return false;
        }

        return orderComponents[0].equals("push");
    }

    private static boolean isFull(int firstIndex, int lastIndex, int arraySize) {
        return firstIndex == (lastIndex + 1) % arraySize;
    }
}
```
