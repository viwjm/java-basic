# 📌 문제 이름

프로그래머스 120820번 - 나이 출력

## 🔗 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/120820

## 💡 문제 해결 전략

입출력 예
age	result
40	1983
23	2000
에서 result = 2023 - age 임을 유추할 수 있다

## 💻 코드

```java
class Solution {
    public int solution(int age) {
        int answer = 0;
        answer = 2023 - age;
        return answer;
    }
}
```
