# 📌 문제 이름

> 프로그래머스 120805번 - 몫 구하기

## 🔗 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/120805?language=java

## 💡 문제 해결 전략

int 형의 나누기는 자동으로 몫이 된다는 것을 사용하는 문제이다

## 💻 코드

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        answer = num1 / num2;
        return answer;
    }
}
```
