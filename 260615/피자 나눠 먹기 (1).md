# 📌 문제 이름

프로그래머스 120814번 - 피자 나눠 먹기 (1)

## 🔗 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/120814

## 💡 문제 해결 전략

피자의 수를 p라고 하자
그러면 조각의 수는 7p이다
7p >= n 을 만족하는 최소의 정수 p를 구하는 문제이다
수학적으로는 p >= n/7 과 동치이다
int 형 계산 특성상 p = n/7이라고 하면 내림을 해버리기 때문에 틀리게 된다
따라서 올림을 구현해야 한다

그냥 mod 로 케이스를 나눠서 하도록 하려 했으나 우리는 원칙적으로는 조건문을 안 배웠으므로 수학적으로 해결해보자
p = n/7 은 n = 7 일 때는 맞다
하지만 n = 1, 2, 3, 4, 5, 6 인 경우는 p = 0 이 나와버려 틀리게 된다
정담은 n/7 + (보정값)이 되어야 한다
이 보정값은 n이 7로 나누어떨어질 때는 0 그외의 경우는 1이어야 한다

n 이 7의 배수일 때만 n%7이 0이고 나머지 경우는 n%7이 0이 아님을 이용하자
p = n/7 + (6 + n%7)/7
이러면 딱 맞는다


## 💻 코드

```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        answer = n/7 + (6 + n%7) / 7;
        return answer;
    }
}
```
